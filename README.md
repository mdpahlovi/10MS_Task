# IELTS Course

A online IELTS course home page design for [10Minuteschool](https://10minuteschool.com/).

## Running Locally

```bash
git clone https://github.com/mdpahlovi/10MS_Task.git
cd 10MS_Task
npm install
npm run dev
```

Or

```bash
git clone https://github.com/mdpahlovi/10MS_Task.git
cd 10MS_Task
docker compose up -d
```

Poject run on [localhost:3000](http://localhost:3000).
